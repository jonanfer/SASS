Ejemplo SASS

CSS por su cuenta puede ser divertido, pero las hojas de estilo son cada vez más grande, más compleja y más difícil de mantener. Aquí es donde un preprocesador puede ayudar. Sass le permite utilizar características que no existen en CSS sin embargo, como las variables, jerarquización, mixins, herencias y otras cosas ingeniosas que hacen que la escritura diversión CSS nuevo.
Una vez que empiece jugando con Sass, que tomará su archivo Sass preprocesado y guardarlo como un archivo CSS normal que usted puede utilizar en su sitio web.

Pasos para trabajar en SASS

  1. >> ruby -v
  2. >> gem -v
  3. >> gem install sass
  4. >> gem install --http-proxy http://proxy2.sena.edu.co:80 sass
  5. >> sass -v
  6. >> sass-convert variables.scss variables.sass
  7. >> sass variables.scss variables.css
  8. >> sass variables.scss variables.css --style compressed
  9. >> sass --watch variables.scss:variables.css
